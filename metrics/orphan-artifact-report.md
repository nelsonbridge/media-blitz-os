# Orphan Artifact Report

## Purpose

Identify artifacts, records, or outputs that risk becoming stored material instead of integrated knowledge.

Orphan control prevents archive drift.

## Orphan Definition

A record is an orphan if it lacks one or more of the following:

1. Source lineage.
2. Proof boundary.
3. Graph relationship.
4. Opportunity evaluation.
5. Capability mapping.
6. Next action or disposition.
