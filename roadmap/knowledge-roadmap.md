# Knowledge Roadmap

## Purpose

The Knowledge Roadmap governs what the Nelson Knowledge System should mature next.

It prevents the system from merely processing whatever source is nearest. Instead, it directs extraction toward strategic knowledge capabilities.

## Roadmap Principle

Mission drives capability. Capability drives extraction. Extraction drives corpus growth. Corpus growth drives publication. Publication feedback enriches the roadmap.

## Current Strategic Objective

Reach Publication Milestone 1 with a flagship article that proves the NKS operating model and creates a reusable foundation for future public assets.

## Internal Coherence Questions

Every new artifact must answer:

1. Which strategic objective does this support?
2. Which capability does this mature?
3. Which existing artifacts strengthen it?
4. Which future artifacts depend on it?
5. Which proof boundary applies?
6. Which visual artifact should represent it?
7. Which publication milestone does it advance?

## Current Rule

Do not process source material merely because it is available. Process the source that most advances the roadmap.
