# Capability Gap Register

## Purpose

Track the highest-value capability gaps preventing the Nelson Knowledge System from maturing from V1 governed knowledge manufacturing into V2 self-assessing platform behavior.

## Scoring

| Severity | Meaning |
|---:|---|
| 5 | Blocks V2 self-assessment. |
| 4 | Weakens corpus health, validation, or release testing. |
| 3 | Limits reuse, graph coverage, or proof confidence. |
| 2 | Improves convenience but is not strategically critical. |
| 1 | Cosmetic or low-leverage. |
