## Thread Draft

1. A chat thread is not a control plane.

2. A folder is not a control plane either.

3. A serious knowledge system needs durable state: rules, schemas, templates, queues, sync status, and execution history.

4. Drive is useful as an editorial workspace.

5. GitHub is useful as a control plane.

6. The workspace contains the work. The control plane governs how the work evolves.

7. Separate the workspace from the control plane.

8. That separation creates continuity when one surface fails.
