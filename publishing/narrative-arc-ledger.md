# Narrative Arc Ledger

## Purpose

```text
Recognition → Reframe → Framework → Proof → Application → Consequence → Invitation
```

## Status Values

- `Arc Needed`
- `Partial Arc`
- `Arc Drafted`
- `Arc Review Needed`
- `Arc Approved`
