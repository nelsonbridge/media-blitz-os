# Archive-to-Corpus Diagnostic

## Purpose
A repeatable assessment that distinguishes passive information storage from an active knowledge manufacturing capability.

## Dimensions
1. Source lineage
2. Canonical artifacts
3. Proof boundaries
4. Relationship density
5. Knowledge graph coverage
6. Opportunity generation
7. Automation readiness
8. Capability growth

## Scoring
Each dimension is scored 0–5.

- 0–10: Archive
- 11–20: Organized repository
- 21–30: Managed knowledge base
- 31–40: Knowledge manufacturing system

## Outcome
The assessment produces prioritized capability gaps rather than publication recommendations.
